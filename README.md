Custom Stomach Storage
[b]Custom Stomach Storage[/b] allows you to customize the types and quantities the Slugcat can store in stomachs.

[b]Configurable Stomach Capacity[/b] - Set how many items a Slugcat can carry in their stomach
[b]Configurable Keys[/b] - Press C to swallow the object in your hand, press V to regurgitate the last swallowed object
[b]Custom Swallow Types[/b] - Choose which creature types and item types can be swallowed
[b]Advanced Grip Control[/b] - Configure how objects are grasped (one hand, large one hand, two hands, dragged, or cannot be grasped)
[b]Spearmaster Support[/b] - Optional setting allowing Spearmaster to store items in their stomach
[b]Multiplayer Mode[/b] - Supports multiplayer mode with independent stomach storage for each player
[b]Visual HUD Display[/b] - Display stomach contents on screen

Thanks to Jimarad's "What's in My Pocket?" source code

自定义胃部存储

[b]自定义胃部存储[/b] 允许你自定义蛞蝓猫可以在胃中存储的类型和数量。

[b]可配置胃容量[/b] - 设置蛞蝓猫胃中可以携带多少物品
[b]可配置按键[/b] - 按C键吞咽手中物体，按V键反刍最后一个吞咽的物体
[b]自定义吞咽类型[/b] - 选择哪些生物类型和物品类型可以被吞咽
[b]高级抓取控制[/b] - 配置物体的抓取方式（单手、大单手、双手、拖动或不能抓取）
[b]矛大师支持[/b] - 可选设置允许矛大师在胃中存储物品
[b]多人模式[/b] - 支持多人模式，每个玩家拥有独立的胃部存储
[b]视觉HUD显示[/b] - 在屏幕上显示胃里的内容
